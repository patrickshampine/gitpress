gitpress
=======

### usage

Install as a plugin or as a submodule.

Use the function to call the plugin in a theme.

```
gitpress();
```

Classes to override `.gitpress-container`, `.gitpress-active` and `.gitpress`. Defaults to pos:abs, bot:0, width:100%.
